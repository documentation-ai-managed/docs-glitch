## Overview

You can import code from GitHub directly into Glitch.

## Steps

1. In the Glitch dashboard, click **New Project** > **Import from GitHub**
2. Enter your GitHub repo URL
3. Click **OK** to import

**Tip:**
- All files (except those in `.gitignore`) will be imported.
- You may need to set up `.env` variables afterwards.
