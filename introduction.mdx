## Overview

Glitch provides your default documentation space to organize and manage all project documentation effortlessly. You create structured docs using MDX, collaborate with teams, and publish professional documentation sites. Start building your knowledge base today with intuitive tools that scale with your projects.

<Callout kind="info">
Glitch supports real-time editing and version history for seamless team workflows.
</Callout>

## Quick Start

Get started with Glitch in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://glitch.com` with your email or GitHub account.
  </Step>
  <Step title="New Doc Space" icon="plus">
    Click "New Space" and choose a template for your project.
  </Step>
  <Step title="Add Content" icon="edit-3">
    Edit pages using the MDX editor and preview changes live.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy your site with one click and share the URL.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Example: Embed Glitch docs in your app
  import { GlitchDocs } from '@glitch/docs-sdk';

  const docs = new GlitchDocs({ apiKey: 'YOUR_API_KEY' });
  docs.render('#my-container', { spaceId: 'your-space-id' });
  ```
  ```python
  # Example: Fetch Glitch docs via API
  import requests

  response = requests.get(
    'https://api.glitch.com/spaces/your-space-id/docs',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
  )
  print(response.json())
  ```
</CodeGroup>
