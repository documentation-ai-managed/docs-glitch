## Recent Updates

Stay informed about Glitch's latest releases. You receive new features, performance improvements, and bug fixes regularly. Check this page for detailed changelogs organized by version.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for documentation editing, allowing multiple users to work simultaneously.
- Introduced version history with diff viewer to compare changes easily.

## Improvements

- Enhanced search functionality with fuzzy matching and syntax highlighting in results.
- Optimized page load times by `>30%` through better asset bundling.

## Bug Fixes

- Fixed markdown parsing issues with nested lists and tables.
- Resolved authentication errors during long editing sessions.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated AI-powered content suggestions to help you draft documentation faster.
- Added custom themes support using your brand color `#3B82F6`.

## Bug Fixes

- Corrected export to PDF functionality that failed on pages with images.
- Patched security vulnerability in public page sharing.

````jsx
// Example: New theme configuration
const themeConfig = {
  primaryColor: "#3B82F6",
  fontFamily: "Inter, sans-serif"
};
````

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched core documentation workspace with MDX support and live preview.
- Implemented user authentication with API key management.

## Breaking Changes

- Updated API endpoints from `/v0/` to `/v1/` prefix. Migrate your integrations.

## Improvements

- Added keyboard shortcuts like `<kbd>Ctrl</kbd>+<kbd>S</kbd>` for quick saves.

</Update>
